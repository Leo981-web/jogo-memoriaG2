# Jogo da Forca TDD — Desafio Guiado por IA

Este repositório **não contém código**. Ele contém as instruções para um agente de IA construir o projeto junto com você, passo a passo.

## Como começar

```bash
git clone <url-deste-repositorio>
cd aula-12-jogo-da-forca-tdd-ai-challenge
```

Abra a pasta no **Claude Code**, **Gemini CLI** ou outro agente compatível com `AGENTS.md` e digite:

```
oi
```

O agente assume o controle a partir daí.

## O que você vai construir

Um **Jogo da Forca multiplayer em tempo real** com:

- Node.js + WebSockets (sem framework)
- Arquitetura em camadas (Domain → Application → Infra)
- TDD com Jest (ciclo Red → Green → Refactor)
- Frontend em Vanilla JS com Tailwind CSS

## Conceitos praticados

- Pirâmide de Testes e Jest (Aula 8)
- Clean Code e nomenclatura (Aula 10)
- SOLID — SRP, DIP, encapsulamento (Aula 11)
- Clean Architecture e padrões Facade e Observer (Aula 13)
